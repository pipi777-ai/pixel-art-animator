# 🎨 Pixel Art Animator

ブラウザで動くピクセルアートアニメーションエディタです。インストール不要、`index.html` をブラウザで開くだけで使えます。

## 機能

- **ピクセル描画** — ペン・消しゴム・塗りつぶし・スポイトツール
- **アニメーション** — フレームの追加・複製・削除、FPS調整、再生プレビュー
- **カラー** — 20色パレット + カスタムカラーピッカー + 不透明度調整
- **ブラシサイズ** — 1〜8px
- **ズーム** — キャンバスの拡大・縮小
- **グリッド表示** — オン/オフ切り替え
- **エクスポート** — 現在フレームをPNG保存 / 全フレームをGIFアニメとして保存
- **Undo / Redo** — 各フレームごとに最大50段階

## 使い方

```
index.html をブラウザ（Chrome / Edge 推奨）で開く
```

## キーボードショートカット

| キー | 操作 |
|------|------|
| `P` | ペンツール |
| `E` | 消しゴム |
| `F` | 塗りつぶし |
| `I` | スポイト |
| `G` | グリッド表示切り替え |
| `+` / `=` | ズームイン |
| `-` | ズームアウト |
| `Ctrl+Z` | 元に戻す |
| `Ctrl+Y` | やり直す |

## キャンバスサイズ

8×8 / 16×16（デフォルト）/ 32×32 / 48×48 から選択できます。  
※サイズ変更時は全フレームがリセットされます。

## 技術スタック

- HTML / CSS / JavaScript（フレームワークなし）
- [gif.js](https://jnordberg.github.io/gif.js/) — GIFエクスポート用
